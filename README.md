This is a console-based receipt generator built using Python for a small shop named **Kiran Store**.

- Add multiple item prices
- Display running total after each entry
- Exit option using `Q`
- Basic input validation
- Final bill display

---

## How to Run
1. Open the Python file.
2. Run the program using Python.
3. Enter item prices one by one.
4. Enter `Q` to generate the final receipt.
